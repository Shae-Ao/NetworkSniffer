# NetworkSniffer
 
  开发一个网络嗅探器，重点对TCP、UDP、ARP、IGMP、ICMP 等数据包进行分析，实现捕捉前过滤、数据包统计、流量统计等功能。

## 技术栈
    MFC + WinPcap V4.1.3 + skin++

## 开发
```bash
    # 克隆项目
    git clone https://github.com/zhenzhencai/NetworkSniffer.git

```
